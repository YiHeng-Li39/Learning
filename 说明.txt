上传新的文件夹：
1.右键该文件夹，以终端打开：git add .
2.对这次上传的东西做个批注：git commit -m "更新说明"
3.保存到云端：git push

删除旧的文件夹：
1.在电脑的 D:\Github 文件夹里，直接把那个不需要的文件或文件夹删掉
2.记录变动：在终端里输入 git add . 并回车。
3.提交变动：输入 git commit -m "删除xxx文件" 并回车。
4.云端同步：输入 git push 并回车。刷新 GitHub 网页。
